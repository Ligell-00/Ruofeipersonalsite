# UI Detail Polish

Use this skill to make the website feel finished and consistent.

## Workflow

1. Read `AGENTS.md` and the relevant HTML/CSS/JS.
2. Check the issue on desktop and mobile when possible.
3. Make small, scoped improvements.
4. Update `design-qa.md` after meaningful visual changes.

## Checklist

- Spacing and rhythm
- Typography and readability
- Navigation states
- Card polish
- Responsive layout
- Dark mode
- Image fit
- Text overflow

## Rule

Reuse existing styles and avoid broad restyles unless asked.
