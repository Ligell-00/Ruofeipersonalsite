# Personal Website Planning

Use this skill to keep personal website planning clear and actionable.

## Workflow

1. Read `AGENTS.md` first when inside the personal website repo.
2. Check the current page structure and `git status`.
3. Clarify the goal of the current round: planning, content, UI, code, or QA.
4. Produce a short roadmap with priorities and acceptance criteria.

## Keep In Mind

- Keep the four current entries: 简历, 产品案例, 交互/UI, vibe coding.
- Prioritize the strongest proof first, especially 掌中宝交易 OMS.
- Keep the site concrete, professional, and evidence-led.
- Do not introduce new tech unless the user asks.
