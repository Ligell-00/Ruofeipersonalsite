# Interaction UI Writing

Use this skill to describe design work as evidence of problem solving.

## Workflow

1. Read `interaction-ui.html` and relevant materials.
2. Identify the design problem, design action, and result.
3. Write concise Chinese sections that connect UI decisions to user or business value.

## Useful Sections

- 核心链路
- 复杂流程降噪
- 设计系统
- 转化页面
- 方法沉淀

## Writing Rule

Avoid saying "会设计"; show what improved because of the design.
